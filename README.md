# hr-analytics-sql-project
HR Analytics and Employee Attrition Analysis using SQL Server
PROJECT OVERVIEW

This project analyzes employee data to identify factors contributing to employee attrition and workforce performance. The analysis provides actionable insights that can help organizations improve employee retention, compensation strategies, and workforce planning.


BUSINESS QUESTIONS

The project answers questions such as:
Which departments have the highest attrition?
Which job roles earn the highest salaries?
Are there high-performing employees who may be underpaid?
Which employees are at higher risk of leaving?
How does attrition vary by gender, education, and marital status?
What workforce KPIs should management monitor?


DATASET

HR Employee Attrition dataset containing information on:
Demographics,
Compensation,
Performance ratings,
Job satisfaction,
Years at company,
Department information,
Attrition status.


SQL SKILLS DEMONSTRATED

SELECT statements,
Filtering with WHERE,
Aggregations using COUNT, AVG, MIN, and MAX,
GROUP BY and ORDER BY,
CASE statements,
Window functions,
RANK(),
PERCENT_RANK(),
SUM() OVER(),
Subqueries,
Correlated subqueries,
Business rule implementation.


KEY FINDINGS

Identified departments with the highest turnover rates.
Built attrition risk classifications for employees.
Detected high-performing employees receiving below-average compensation.
Produced executive KPI reports for workforce planning and decision-making.


BUSINESS RECOMMENDATIONS

Prioritize retention initiatives in high-turnover departments.
Review compensation for high-performing employees.
Monitor employees classified as high attrition risk.
Track workforce KPIs regularly to support strategic HR decisions.
